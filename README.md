## Ejecutar localmente

### Backend (Django REST Framework)

#### 1. Entrar a la carpeta del backend
```bash
cd backend
```
#### 2. Crear y activar un entorno virtual
**Linux / macOS**
```bash
python3 -m venv venv
source venv/bin/activate
```
**Windows (PowerShell)**
```bash
python -m venv venv
venv\Scripts\activate
```
#### 3. Instalar dependencias
```bash
pip install -r requirements.txt
